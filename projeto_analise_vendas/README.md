# 📊 Projeto SQL — Análise de Vendas e Custos

## 🎯 Objetivo
Analisar o custo dos pedidos por estado, considerando regras de negócio como aumento de custo e filtros por tipo de produto, com o objetivo de gerar insights para tomada de decisão.

---

## 🧱 Base de dados
Tabelas utilizadas:
- cap10.dsa_pedidos
- cap10.dsa_clientes
- cap10.dsa_produtos

---

## 📊 Análises realizadas

### 🔹 1. Exploração inicial
Consulta e entendimento dos dados.

### 🔹 2. Métricas gerais
Cálculo de custo total por estado.

### 🔹 3. Análise consolidada
Aplicação de regras de negócio:
- Aumento de custo para o estado de SP
- Filtro por tipo de produto
- Classificação de custo
- Identificação de status de aumento

---

## 🧠 Técnicas utilizadas
- JOIN (INNER JOIN)
- CASE WHEN
- GROUP BY
- HAVING
- LIKE
- BETWEEN
- Funções de agregação (SUM)

---

## 🚀 Insights possíveis
- Estados com maior custo de pedidos
- Impacto de aumento de custo em regiões específicas
- Identificação de padrões por tipo de produto
