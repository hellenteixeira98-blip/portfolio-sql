# 📊 Projeto: Análise de Clientes
## 🎯 Objetivo
Analisar o comportamento de clientes e entender padrões de compra.
## 📌 Análise realizada
Foi criada uma query para identificar os clientes mais valiosos com base em:
- Quantidade de pedidos
- Faturamento total
- Valor médio de compras
- Ranking de clientes
