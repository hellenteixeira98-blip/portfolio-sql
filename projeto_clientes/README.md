# 📊 Projeto: Análise de Clientes

## 🎯 Objetivo
Analisar o comportamento de clientes com base no histórico de compras, identificando frequência, faturamento, ticket médio e o Top 10 clientes com maior faturamento.

---

## 📁 Base de dados

Tabela: `clientes`

Contém:
- id_cliente
- nome
- data_compra
- valor_compra

---

## 🔧 Análise realizada

Foi desenvolvida uma query em SQL para calcular:

- Quantidade de pedidos por cliente  
- Faturamento total por cliente  
- Ticket médio por cliente
- Top 10 faturamento

---

## 💻 Query

A query utilizada está disponível no arquivo:
👉 `analise.sql`

---

## 📊 Insights (resumo)

- Clientes com maior faturamento se destacam na base  
- Existe variação no ticket médio entre clientes  
- Nem sempre quem compra mais vezes é quem mais gera receita  

---

## 📌 Conclusão

A análise permite identificar clientes mais valiosos e entender padrões de consumo, auxiliando estratégias de retenção e segmentação.
